
# FAQ

## Why is a plugin marked as "Unknown"?

Some plugins do not declare compatible Redmine versions.
In such cases the plugin performs only static analysis.

## Why is startup not enough?

Many plugins successfully start but fail during actual use.
Always verify core functionality after upgrading.

## Why does the AI recommend searching for forks?

Many abandoned plugins continue to be maintained by community forks,
which often provide better compatibility than patching old code yourself.
