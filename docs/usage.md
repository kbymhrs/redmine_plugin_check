# Usage

1. Login as an administrator.
2. Open **Administration → Plugin Check**.
3. Enter the target Redmine version.
4. Review the compatibility report.
5. Filter by Risky, Warning, Unknown or OK.
6. Export CSV if required.
7. Export AI Markdown if required.
8. Run AI analysis if configured.

The report is intended as a decision support tool before upgrading Redmine.
