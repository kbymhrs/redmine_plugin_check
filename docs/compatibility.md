# Compatibility Notes

The plugin has been developed while performing actual Redmine upgrades.

Verified upgrade paths include:

- Redmine 3.3 → 4.2
- Redmine 4.2 → 5.1
- Redmine 5.1 → 6.1
- Redmine 6.1 → 7.0

The diagnostic rules continue to evolve as additional migration experience is gained.

Compatibility results should always be confirmed by testing.
