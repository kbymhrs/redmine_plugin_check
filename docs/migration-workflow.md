# Recommended Migration Workflow

Based on real-world migrations from Redmine 3.3 to 7.x.

## Step 1

Scan installed plugins.

## Step 2

Search for:

- maintained forks
- latest releases
- migration guides

## Step 3

Prefer maintained forks over patching abandoned plugins.

## Step 4

Upgrade plugins.

## Step 5

Run plugin migrations.

## Step 6

Start Redmine.

## Step 7

Verify actual functionality.

Recommended checks:

- plugin settings
- create
- edit
- delete
- permissions
- background jobs
- UI behavior
